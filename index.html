• HTML (HyperText Markup Language) é a linguagem de marcação usada para criar páginas da web. Ele serve para estruturar o conteúdo de um site, como títulos, parágrafos, imagens, links e listas. Basicamente, é o “esqueleto” de uma página na internet.

  • JavaScript é uma linguagem de programação usada para tornar páginas da web interativas. Com ele, é possível criar efeitos visuais, validar formulários, atualizar conteúdos sem precisar recarregar a página, criar jogos, aplicativos web e muito mais.

  • Python é uma linguagem de programação versátil e de fácil aprendizado. É usada em desenvolvimento web, análise de dados, inteligência artificial, automação de tarefas, ciência de dados e muito mais. É conhecida por sua sintaxe simples e clara.

  • MySQL é um sistema de gerenciamento de banco de dados relacional. Ele permite armazenar, organizar e consultar dados de forma eficiente. É muito usado em sites e aplicativos para gerenciar informações como usuários, produtos, vendas e mensagens.
